BLI MEDLEM

HUR?

För medlemskap krävs ett giltigt studieintyg från ditt lärosäte. Med ett sådant intyg och giltig legitimation till hands kan du glatt knalla till någon av våra landsmannaförmän eller curatorer för att ansöka till nationen. Dom kan även svara på vidare frågor om medlemskapet.

Ni kan även ansöka om medlemskap genom att maila oss på info snabela rosenboms punkt se. Kom ihåg att bifoga ditt studieintyg.

Studieintyg?

Antagningsbesked och registreringsintyg räknas som giltigt studieintyg. Ni kan hämtas ut en sådan på LADOK eller studentcentrum. Som student på Blekinges Tekniska Högskola kan registreringsintyget hämtas här.
